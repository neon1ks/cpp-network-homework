Для генерации приватного ключа (*.key) и сертификата (*.pem) используйте команду:
openssl req -newkey rsa:2048 -nodes -keyout sslserver.key -x509 -days 3650 -out sslserver.pem
При вводе поля Common Name (e.g. server FQDN or YOUR name) [] обратите внимание, что здесь надо
ввести имя сервера или его IP-адрес в нашем случае это 127.0.0.1

Для просмотра данных сертификата использовать команду
keytool -printcert -file ./sslserver.pem

Для простоты эксперимента предолженные файлы *.key и *.pem скопировать в папку с исполняемым файлом сервера, и
*.pem скопировать в папку с исполняемым файлом клиента, для установления защищенного соединения.
